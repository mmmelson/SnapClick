# SnapClick v2.0 发布说明

## 🎉 正式版发布

我们很高兴地宣布 SnapClick v2.0 正式发布！这是一款专为 macOS 设计的现代化鼠标连点器，采用原生 SwiftUI 构建，提供流畅、优雅的使用体验。

## ✨ 主要特性

### 🎯 多方案管理
- 创建和管理多个独立的连点方案
- 每个方案可自定义名称、按键类型、点击次数和时长
- 支持快速启用/禁用方案
- 一键全开/全关所有方案

### ⚡ 全局快捷键
- 为每个方案设置独立的快捷键
- 支持 Command、Option、Control、Shift 修饰键组合
- 可视化快捷键录制界面
- 全局监听，任何时候都能触发

### 🖱️ 精确点击控制
- 支持左键和右键连点
- 可自定义点击次数（1-999+）
- 精确设置完成时长（秒）
- 自动计算点击间隔

### 🎨 现代化界面
- 采用 SwiftUI 原生框架
- 640×450 紧凑窗口设计
- 固定宽度布局，无跳动
- 菜单栏常驻图标

### 📊 实时反馈
- 方案启用/禁用状态实时显示
- 执行连点时显示系统通知
- 每次点击伴随音效反馈
- 清晰的方案计数显示

## 📋 系统要求

- **操作系统**: macOS 13.0 (Ventura) 或更高版本
- **处理器**: Apple Silicon (M1/M2/M3/M4)
- **权限**: 需要授予辅助功能权限

## 📦 安装说明

### ⚠️ 重要：首次打开

由于应用未经 Apple 公证，首次打开时需要：

1. **右键打开**（推荐）
   - 在 Finder 中找到 `SnapClick.app`
   - 按住 Control 键点击应用
   - 选择「打开」→ 确认「打开」

2. **命令行方法**
   ```bash
   xattr -cr /Applications/SnapClick.app
   open /Applications/SnapClick.app
   ```

3. **系统设置方法**
   - 双击应用（会被阻止）
   - 前往「系统设置」→「隐私与安全性」
   - 点击「仍要打开」

### 🔐 授予权限

应用需要「辅助功能」权限来监听快捷键和模拟点击：

1. 启动应用后，尝试启用任一方案
2. 系统会提示需要权限
3. 前往「系统设置」→「隐私与安全性」→「辅助功能」
4. 勾选 SnapClick

## 🚀 快速开始

1. **创建方案**
   - 点击「新增」按钮
   - 填写方案名称
   - 选择鼠标按键（左键/右键）
   - 设置点击次数和时长
   - 录制快捷键
   - 保存方案

2. **启用方案**
   - 点击方案右侧的圆圈图标
   - 图标变为绿色 ✅ 表示已启用

3. **使用方案**
   - 将鼠标移到目标位置
   - 按下设置的快捷键
   - 自动执行连点

## 💡 使用技巧

### 点击间隔计算
```
点击间隔 = 完成时长 ÷ 点击次数

示例：
- 10 次 / 1.0 秒 = 0.1 秒间隔（100ms）
- 50 次 / 5.0 秒 = 0.1 秒间隔（100ms）
- 20 次 / 2.0 秒 = 0.1 秒间隔（100ms）
```

### 快捷键推荐
- ✅ 使用 **⌘⌥ + 字母键**（如 ⌘⌥A）
- ✅ 避免系统快捷键（⌘C、⌘V、⌘Q 等）
- ✅ 为不同场景设置不同组合

### 最佳实践
1. **先测试** - 在安全区域测试方案
2. **合理命名** - 使用描述性名称
3. **定期备份** - 备份 schemes.json
4. **及时禁用** - 不用时禁用方案

## ⚠️ 已知限制

1. **仅支持 Apple Silicon**
   - 当前版本为 arm64 架构
   - Intel Mac 需要等待 Universal 版本或自行编译

2. **无 Apple 公证**
   - 首次打开需要手动授权
   - 未来版本将提供公证版本

3. **系统负载影响**
   - 点击间隔精度受系统负载影响
   - 建议设置合理的时长（≥0.5 秒）

4. **应用兼容性**
   - 某些应用/游戏可能检测并阻止自动点击
   - 请遵守相关应用的使用条款
